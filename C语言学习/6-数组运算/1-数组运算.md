### 数组的集成初始化

```c
int a[] = {1,2,3};
```

我们在 [] 中没有输入值，编译器帮我们来数。

如果`int a[13] = {2};` 除了a [0] 为 2，以外都是 0

* 定位

  ```c
  int a[10] = {
      [0] = 2, [3] = 3,4,
  };
  ```

  * 接在后面没有定位的数据，直接为下一个数据
  * 其他位置补上 0
