## 运算优先级

| 优先级 | 名称     | 运算符 | 结合性 |
| ------ | ---------- | ------- | ------ |
| 1      | 括号     | ()      | ->     |
| 2      | 单目运算符 | ! ++ -- | <-     |
| 3      | 乘除     | * / %   | ->     |
| 4      | 加减     | + -     | ->     |
| 5      | 比较大小 | < > =>  | ->     |
| 6      | 判断     | == !=   | ->     |
| 7      | 逻辑     | && ``||``   | ->     |
| 8      | 赋值     | =       | <-     |
