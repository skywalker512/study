## 逻辑类型
最原始的 C 中没有布尔类型，后面加入进来 \
需要引入```#include <stdbool.h>``` 就可以使用 true false \
注意：bool 依旧是 int，但不能被赋值只能是1或0(大于1的数等同与1)
```
#include <stdio.h>
#include <stdbool.h> // 新加入的

int main() {
    bool t = 6 > 5;
    printf("%d\n", t);  //输出为 1
    t = false;
    printf("%d\n", t);  //输出为 0
    t = 8;
    printf("%d\n", t);  //输出为 1
    t = 1;
    printf("%d\n", t);  //输出为 1
    t = 0;
    printf("%d\n", t);  //输出为 0
    return 0;
}
```
